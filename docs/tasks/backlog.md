- **現状**: 収益化なしでリリース優先
- **議論まとめ（2026-02）**:
  - フリーミアム課金は実装コストが高い（課金画面、購入状態管理、リストア機能など）

- **将来の選択肢**:
  - フリーミアム（エクスポート機能、検索強化など）
  - バナー広告（検索・本一覧など整理フェーズ画面のみ）
