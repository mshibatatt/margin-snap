## サブタイトル（30文字以内）
読書の気づきを逃さない

## 説明文

本を読んでいる時の「ここ大事」をすぐに記録できるアプリです。

【特徴】

- 写真かページ番号で記録
  本のページを撮影するか、ページ番号を入力するだけで記録できます。

- スタンプや一言メモも添えられる
  感情スタンプや短いメモを添えて、その時の気持ちも残せます。

- 後から本ごとに整理
  記録したメモは、後からまとめて本に紐づけられます。読書中は記録に集中できます。

【こんな方におすすめ】

- 読書中に気になった箇所をすぐにメモしたい
- 付箋やノートを用意するのが面倒
- 読み返したい箇所を忘れてしまう
- 読書の感想を手軽に残したい

シンプルな操作で、読書体験をより豊かにします。
